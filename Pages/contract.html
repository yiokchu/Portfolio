CONTRACT MANAGEMENT



Collaboration with Josephine Chan, Kaiyin Chen, and Lily Kiziriya


The DYCD contract management website organizes the function of the contact management for users to find based on need. The task was to create a FAQ/ Help/ Tutorials page for an internal website system used by DYCD employees. Since the data is confidential the information has been substituted.

The website was created from scratch using HTML and CSS, and hosted on Github Pages.

Main Contact: Lenny Dorodney, Deputy Director and Vladimir Moldovan, .Net Developer

Take a LOOK at the Website

Functionalities:


  1. Direct users on how to use the tabs for their intended purpose and create an FAQ archive for immediate assistance
  2. Suggests users to submit tickets for any requested help not already listed in order for us to better understand the needs of our team


Features:


  1. Drop Down Aside Navigation - to index the different pages in Contract Managemen
  2. Tagged Navigation - inner navigation buttons are tagged to automatically bring the viewport to the desired section of text
  3. Minimalist Design - designed to look comfortable to the human eye in any window size
  4. Adaptive Design- minimalist design with a light blue theme so as not to distract from the functionality of the webpage
