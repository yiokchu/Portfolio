RemDose

Description
RemDose is a pill reminder app that keeps people
on track of their medication and prescriptions with
daily reminders. This app helps people begin forming
the healthy habit of adhering to their medication
schedule and offers a practical and convenient solution
for the problem of forgetting to take the pills.
Tools
JavaScript
Figma
React Native
Role
Full Stack Developer and Sole Designer
Process
Wireframe: I started by laying out a basic wireframe
with the functionalities I wanted to add to each page.
Functionalities:
  • Track of Medication
  • Daily Reminders
    1. Add/Delete/Edit Medication
      1. Name
      2. Quantity
      3. Dose
  • Alarm
    1. Audio
    2. Push Notifications
Prototype: Once I had the desired layout I decided
to create a prototype. I needed to carefully consider challenges with the User Experience since the targeted audience would also include people 65 and above.
It was key that the UI was simple, friendly, and easy to
use and understand. To accomplish this, I implemented
the following design guidelines in my app:
  • Make the User Interface big and minimal
  • Use a legible and large font (the minimum font
    size used is 16px)
  • Include space between items
  • Label Icons
Typography
Main Color Palette
Secondary Color Palette
Icons
Logo and Name
Process
Programming: With the prototype done I started to
code the app using React Native which allowed me to simultaneously develop for iOS and Android while maintaining the appearance, feel, and productivity.

React Native gave me access to features along with
access to several integrated core components. I was
able to implement the alerts to users through push notifications by creating different channels that
create a unique token for each notification.
In addition, I implemented a circular progress bar that updates every time a medication is checked.
Results
The app allows the user to create recurring reminders
and notify them when the medication is due to be taken.
I added features like a checkbox to indicate if the medication has been taken as well as a daily progress
bar, so the user has more than one way to identify if
they had taken their medication. The user can add a
new medication or delete or edit an existing one.
After saving, the application will display the medication
details including the medicine name, dosage,
quantity, and time. Then RemDose is able to send push
notifications at certain times to remind the user to
take their medication.

People using RemDose can form the habit of adhering
to their dosage schedules which will improve their treatments and recovery.
