Covid App

Description
The Covid App is a tool used to keep the
user up to date with the current COVID-19
information. This app helps people be
prepared by staying informed.
Tools
Figma
Adobe Illustrator
Role
Sole Designer
Timeline
30 minutes
Process
I designed the user interface using Figma.
The minimalist app keeps the information as simple
and straightforward as possible.
Functionalities:
  • Display the number of COVID-19 cases globally
  • Chart Display of the number of cases
  • Display by state: Number of Cases, Number of Deaths
  • Symptoms list
  • Quarantine information
  • Rapid Test Locations
  • COVID Vaccine information


Wireframe: I first created a wireframe on
paper to lay out the content and functionality
on a page and establish the basic structure of the
app before the visual design and content were added.
Prototype: Once I had the desired layout I
decided to create a prototype using Figma. The
information is kept as concise and straightforward
as possible by using a minimalist interface.
