Calendar App

Description
The Calendar App is a tool that helps the
user to visualize their schedule and remind
them of important events. This app is easy
to use since it requires minimal clicking
and menu exploration.
Tools
JavaScript
React Native
Role
Sole Designer and Full Stack Developer
Timeline
45 minutes
Process
I designed the user interface using React Native.
The minimalist app keeps the information as simple
and straightforward as possible.
Functionalities:
  • Event Scheduling
  • Monthly Calendar Display
  • Weekly Calendar Display
  • Event Display


Wireframe: I started by laying out a
basic wireframe with the functionalities I wanted
to add to each page.

Then, I started to code the app using JavaScript
and React Native. I used predefined icons from
the IonIcons library, and selected a simple color
palette of Blue and Orange.
