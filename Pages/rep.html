AUTO REPAIR

This website was made as a simple information page for the L.L.L Auto repair shop. The objective of this page was to display details, thus the customers could have quick access to the contact information, location of the shop, as well as the auto repair's hours and services.

The website was created from scratch using HTML and CSS, and hosted on Netlify.

Features:


  1. Swipe Effect - Scrolling images that fill the viewport size and snap at the start so it gives a swipe effect
  2. Static Buttons - Designed to make emphasis on the swipe effect
  3. Map - Embedded map to have quick access to the location of the shop
  4. Minimalist - Designed to not distract from the functionality of the webpage
