COVID APP

The Covid App is a tool used to keep the user up to date with the current COVID-19 information. This app helps people be prepared by staying informed.

I designed the user interface using Figma. The minimalist app keeps the information as simple and straightforward as possible.


Functionalities:



  1. Display the number of COVID-19 cases globally
  2. Chart Display of the number of cases
  3. Display by state: Number of Cases, Number of Deaths
  4. Symptoms list
  5. Quarantine information
  6. Rapid Test Locations
  7. COVID Vaccine information
