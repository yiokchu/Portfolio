Contract Management

Description
The DYCD contract management website organizes the
function of contact management for users to find based
on need. The task was to create a FAQ/ Help/ Tutorials
page for an internal website system used by DYCD
employees. Since the data is confidential the
information has been substituted.
Tools
HTML
CSS
Hosted on Github Pages
Role
Designer and Co-Developer
Functionalities:
  • Direct users on how to use the tabs for their
    intended purpose and create an FAQ archive
    for immediate assistance
  • Suggests users to submit tickets for any
    requested help not already listed in order for
    us to better understand the needs of our team
Features:
  • Drop Down Aside Navigation - to index the different
    pages in Contract Management
  • Tagged Navigation - inner navigation buttons are
    tagged to automatically bring the viewport to the
    desired section of text
  • Minimalist Design - designed to look comfortable
    to the human eye in any window size
  • Adaptive Design - minimalist design with a light
    blue theme so as not to distract from the
    functionality of the webpage
Main Contact:
Lenny Dorodney Deputy Director
Vladimir Moldovan .Net Developer
Process
Stage 1: I started by creating the first wireframe
and site map and coding the site accordingly. At first,
we used the same colors as the NYC-DYCD Department
had on other websites.
Colors
Client's Feedback: "Thanks for the website design draft;
I think you’ve done hard work and achieved beautiful
results! However, I would like to point out a couple
of things. I would like only a side navigation and the
top bar to have the title of the page. As with the
colors, I would also like to have it light blue and
simple to go with the inspiration image..."
Colors
Logo
Stage 2: Edit the wireframe following the
client's feedback.
Stage 3: Edit the wireframe and website to
improve the design and make a better User Experience,
giving the side-nav a dropdown menu, as well as
making the side-nav the whole page height.
Results
